# autonomia

## Git clone
```bash
git clone git@github.com:aut0nomia/autonomia.git
cd autonomia/
```

## Prerequisities

- Python 3.13 or newer
- UV installed

### Install UV

```bash
curl -LsSf https://astral.sh/uv/install.sh | sh

```


## Install dependencies

```bash
uv sync
```

## Run program

```bash
uv run main.py
```
